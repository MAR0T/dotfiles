
# Installation

## Window app
```sh
sudo pacman -S bitwarden
```

## CLI tool
```sh
sudo pacman -S bitwarden-cli
```

## Firefox plugin
[Firefox addon](https://addons.mozilla.org/en-US/firefox/addon/bitwarden-password-manager/)
